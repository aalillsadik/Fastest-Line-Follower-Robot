
# Fastest-Line-Follower-Robot

This is a PID Control line follower robot with full customization and settings. 

**Features**

- It is able to follow any lines on a surface. ( White line on a black surface or White suface on a black line )
- There is a 8 channel IR Sensor to detect the lines on a surface
- Here the front ultrasonic sensor will help you to avoid objects. If it sense any object during the run it will overpass the object and back to th track.
- Its also able to follow the walls. When there will be no lines then it will follow the walls untill its getting the line. It will measure the both distance using the both side ultrasonic sensor. 
- You will be able to calibrate and change any values that need to be change like the pwm signal for the each motors and and so on from the display settings using rotary encoder and push button.
